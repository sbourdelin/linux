*ADT7475 hwmon sensor.

Required properties:
- compatible: One of
	"adi,adt7473"
	"adi,adt7475"
	"adi,adt7476"
	"adi,adt7490"

- reg: I2C address

optional properties:

- invert-pwm: This configures the PWM to use logic low for 100% duty cycle.

Example:

hwmon@2e {
	compatible = ;
	reg = <0x2e>;
	invert-pwm;
};
